Quick Example

Creating a Bell state has never been easier:

from quantiq import QuantumCircuit

# Create entangled Bell state
qc = QuantumCircuit(2)
qc.h(0)        # Put first qubit in superposition
qc.cx(0, 1)    # Entangle it with second qubit
qc.measure_all()

# Run and visualize
result = qc.run(shots=1000)
print(qc.draw())
print(result)

That's it! Three lines of code to create quantum entanglement.
