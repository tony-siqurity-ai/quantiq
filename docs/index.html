Quantum circuits, simplified. A developer-first quantum computing platform.

Quick Start

pip install cirquit

Basic Example

from cirquit import QuantumCircuit

# Create a simple circuit
qc = QuantumCircuit(2)
qc.h(0)
qc.cx(0, 1)
qc.measure_all()

# Run the circuit
result = qc.run()
print(result)
